# Machine Learning Practice

A collection of Machine Learning, Data Science, and Analytics projects built using Python and popular ML libraries. This repository documents my learning journey through supervised learning, unsupervised learning, data analysis, model evaluation, and real-world datasets.

## Projects

### 1. Heart Disease Prediction

* Built classification models to predict the likelihood of heart disease.
* Applied data preprocessing, feature scaling, and model evaluation techniques.
* Implemented and compared multiple machine learning algorithms.

**Tech Stack:** Python, Pandas, NumPy, Scikit-Learn

---

### 2. Urban Heat Island Analysis

* Analyzed Urban Heat Island (UHI) effects using environmental indicators.
* Explored relationships between Land Surface Temperature (LST), NDVI, NDBI, and NDWI.
* Performed visualization and statistical analysis on satellite-derived datasets.

**Tech Stack:** Python, Pandas, Matplotlib, Seaborn

---

### 3. Linear Regression

* Implemented Linear Regression for predictive analytics tasks.
* Performed exploratory data analysis and model evaluation.
* Studied the impact of different features on prediction accuracy.

**Tech Stack:** Python, Scikit-Learn, Pandas

---

### 4. Kaggle Irrigation Classification

* Participated in a Kaggle-style machine learning classification challenge.
* Built and compared Decision Tree, Random Forest, and LightGBM models.
* Generated prediction submissions for evaluation.

**Tech Stack:** Python, Scikit-Learn, LightGBM, Pandas

---

### 5. Telecom Churn Prediction

* Predicted customer churn using classification techniques.
* Implemented K-Nearest Neighbors (KNN) and Naive Bayes models.

**Tech Stack:** Python, Scikit-Learn, Pandas

### 6. Advertising Sales Classification

* Converted sales prediction into a classification problem.

**Tech Stack:** Python, Scikit-Learn, Pandas

### 7. VHI Drought Analysis
